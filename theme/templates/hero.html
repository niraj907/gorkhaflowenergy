Gorkha Flow (4.06 MW)

Developed by: Gorkha Flow Ltd

GorkhaFlow Hydropower Project

GorkhaFlow Hydropower Project is a cascade-type hydropower project that utilizes water from the outlet of the Upper Chirkhwa Hydropower Project. Located in Sadananda Municipality, Bhojpur District, Province 1, this project efficiently harnesses water resources for power generation.
